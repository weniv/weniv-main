# Button

<Controls />

## Installation

`@weniv/components-button` 패키지를 설치합니다.

<Source
  dark
  language="bash"
  code={`
    yarn add @weniv/components-button
  `}
/>

## Usage

1. `Button` 컴포넌트를 불러옵니다

<Source
  dark
  language="jsx"
  code={`
import { Button } from '@weniv/components-button';
`}
/>

2. `@weniv/components-button/style.css`를 불러옵니다.

<Source
  dark
  language="jsx"
  code={`
    import '@weniv/components-button/style.css';
  `}
/>

3. 버튼의 `variant`를 지정해줍니다.

- `solid`(기본값), `outline`, `outlineSolid`, `text`

<Source
  dark
  language="jsx"
  code={`<Button variant="outline" children="button"  />`}
/>

3. 버튼의 `size`를 지정할 수 있습니다.

- `xs`, `ms`, `md`(기본값), `lg`

<Source
  dark
  language="jsx"
  code={`<Button variant="outline" children="button"  size="lg" />`}
/>
