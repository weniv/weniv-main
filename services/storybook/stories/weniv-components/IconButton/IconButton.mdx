# IconButton

<Primary />
<Controls />

## Installation

`@weniv/components-button` 패키지를 설치합니다.

<Source
  dark
  language="bash"
  code={`
    yarn add @weniv/components-icon
  `}
/>

## Usage

1. `IconButton` 컴포넌트를 불러옵니다

<Source
  dark
  language="jsx"
  code={`
import { Icon as _Icon } from '@weniv/components-icons';
`}
/>

2. `@weniv/components-button/style.css`를 불러옵니다.

<Source
  dark
  language="jsx"
  code={`
    import '@weniv/components-button/style.css';
  `}
/>

3. 아이콘의 타입을 지정해줍니다.

<Source dark language="jsx" code={`<IconButton name="alert" />`} />

4. 아이콘의 너비, 높이 색상을 지정할 수 있습니다.

<Source
  dark
  language="jsx"
  code={`<Icon name="alert" width="10rem" height="10rem" color="error" />`}
/>
