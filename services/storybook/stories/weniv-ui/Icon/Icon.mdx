# Icon

<Primary />
<Controls />

## Installation

`@weniv/components-icon` 패키지를 설치합니다.

<Source
  dark
  language="bash"
  code={`
    yarn add -D @weniv/components-icon
  `}
/>

## Usage

1. `Icon` 컴포넌트를 불러옵니다

<Source
  dark
  language="jsx"
  code={`
import { Icon } from '@weniv/components-icons';
`}
/>

2. `@weniv/components-icons/style.css`를 불러옵니다.

<Source
  dark
  language="jsx"
  code={`
    import '@weniv/components-icons/style.css';
  `}
/>

3. 아이콘의 이름을 지정해줍니다.

<Source dark language="jsx" code={`<Icon name="alert" />`} />

4. 아이콘의 너비, 높이 색상을 지정할 수 있습니다.

<Source
  dark
  language="jsx"
  code={`<Icon name="alert" width="10rem" height="10rem" color="error" />`}
/>

## 아이콘 목록

- 클릭시 코드가 복사됩니다.

<div className="preview_wrap" style={{ display: 'grid', gridTemplateColumns: 'repeat(auto-fill, minmax(7.5rem, auto))', gap: '1em' }}>

    }}

  </button>
</div>
)) }

</div>
