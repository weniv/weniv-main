# Color

## Installation

`@weniv/weniv-css` 패키지를 설치합니다.

<Source
  dark
  language="bash"
  code={`
    yarn add -D @weniv/weniv-css
  `}
/>

## Usage

1. `@weniv/components-icons/style.css`를 불러옵니다.

<Source
  dark
  language="jsx"
  code={`
import '@weniv/components-icons/style.css';
`}
/>

2. css 사용방법

<Source
  language="css"
  code={`
color: var(--primary);
color: rgba(var(--primary-rgb), 0.2);
`}
/>
